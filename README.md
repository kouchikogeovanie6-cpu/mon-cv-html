# 🚀 Projet IMeN - Écosystème de Développement & CV Sémantique

Bienvenue sur le dépôt officiel du projet **IMeN** (Ecosystem Generator), développé dans le cadre de mon parcours d'évaluation technique (2026).

Ce projet démontre l'application de normes industrielles strictes à travers l'automatisation d'un espace de travail et la conception d'une interface web sémantique et accessible.

---

## 🛠️ Architecture & Technologies

L'intégralité du projet a été construite et industrialisée en utilisant les technologies suivantes :

* **Environnement Système :** Linux Ubuntu via WSL (Windows Subsystem for Linux) pour la gestion des scripts et des paquets.
* **Automatisation (Méthode FORGE) :** Scripting Bash pour l'industrialisation et la configuration de l'espace de travail.
* **Langages Front-End :** HTML5 (Sémantique stricte), CSS3 et JavaScript.
* **Gestion de Version & IDE :** Visual Studio Code, Git & GitHub.

---

## 🎯 Fonctionnalités Clés du CV

1.  **Sémantique HTML5 Pure :** Utilisation rigoureuse des balises sémantiques (`<header>`, `<main>`, `<aside>`, `<section>`, `<address>`) pour garantir un référencement et une accessibilité optimaux.
2.  **Assurance Qualité (QA) :** Code 100% valide, testé et certifié sans erreur par le **Validateur W3C** (Bandeau vert obtenu).
3.  **Connectivité Professionnelle :** Intégration sécurisée (`rel="noopener"`) de liens interactifs vers mes réseaux professionnels (LinkedIn, GitHub) et messageries instantanées (WhatsApp direct via l'API, Facebook).
4.  **Déploiement Continu :** Hébergement et mise en production automatisés via **GitHub Pages**.
